# Test Bank Transfer Restful API

This is a simple Restful API services project provisioned to perform funds transfer transactions between accounts.

Used the spring boot framework, Hibernate ORM with H2 in memory database to manage data.

Used lombok to auto-generate constructors/getters/setters for my entity classes, 

Used Springfox for Swagger2.



* [BankTransfer Restful API Swagger UI](http://localhost:8080/swagger-ui.html/)


## Test Bank Transfer API endpoints are: 
* [Get All Accounts](http://localhost:8080/api/bankTransfer/getAllAccounts)

* [Get All Transactions](http://localhost:8080/api/bankTransfer/getAllTransactions)

* [Perform An Inter-Account Transfer](http://localhost:8080/api/bankTransfer/performTransfer)
